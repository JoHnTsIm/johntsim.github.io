CharCounter (A Character Counter Program)

This is a simple program created with python 3.9, that takes a text and returns the length of it. Has a built in user interface as well, so its really easy to work with.

To install it, just press "Download Now" and its gonna just automatically download the setup of the program. Then to install it, just run the setup follow the steps, check the box "Create a desktop shortcut" for easier accces to the program if preferred. if setup ends, simple double click to the program's shortcut on the desktop to start it.
