Windows 11 - Τι ανακοίνωσε η Microsoft

Γιάννης Τσιμητρέας - 29/06/2021

Αρχικά πρέπει να αναφερθεί οτι τα Windows 11 θα είναι μια Δωρεάν Αναβάθμιση των Windows 10, δηλαδή όσοι έχετε ήδη εγκατεστημένα τα Windows 10 στον υπολογιστή σας, θα μπορέσετε πολύ απλά να τα αναβαθμίσετε σε Windows 11, χωρίς κανένα κόστος.

Εμφανισιακό κομμάτι

- Τα εικονίδια της γραμμής εργασιών βρίσκονται στο κέντρο, το ίδιο και το Μενού Έναρξης

- έχουμε επίσης μικρές βελτιώσεις στην φωτεινή/σκοτεινή λειτουργία.

- Στρογγυλεμένες γωνίες σε όλα τα παράθυρα πλέον.

- Καινούρια εφέ μετάβασης

- Καινούρια διάφανα θόλα παράθυρα

- Kαινούρια θέματα για την επιφάνεια εργασίας των Windows

Λειτουργικό κομμάτι

- 'Τα πιο ασφαλή Windows μέχρι τώρα'

- 'Τα Windows 11 είναι γρηγορότερα, το άνοιγμα μετά από αναστολή είναι γρηγορότερο, η αναγνώριση Windows Hello είναι γρηγορότερο, περιήγηση σε οποιονδήποτε Browser είναι γρηγορότερη'

- Βελτιώμενη λειτουργία διαχώρισης παραθύρων (Snap Layout). μέχρι και έως 4 παράθυρα στην ίδια οθόνη με δυνατότητα αποθήκευσης σε περίπτωση αφαίρεσης μιας οθόνης, με σκοπό την επαναφορά των παραθύρων όπως ακριβώς ήταν πριν. Επίσης δυνατότητα προσαρμογής των παραθύρων στις προτιμήσεις μας, με έως και 6 επιλογές, πηγαίνοντας στο κουμπί μεγιστοποίησης του παραθύρου. Σε περίπτωση ανοίγματος νέου παραθύρου, αποθήκευση των παραθύρων ομαδοποιημένα, όπως ακριβώς είναι, με σκοπό την εύκολη και άμεση επαναφορά τους, άμα τα χρειαστούμε και πάλι. Τέλος υπάρχει επιλογή επαναφοράς ομαδοποιημένων παραθύρων ή ενός παραθύρου μόνο.

- Πολλαπλές επιφάνειες εργασίας με δυνατότητα αλλαγής θέματος στην κάθε μια από αυτές

- Windows Updates 40% μικρότερα και πιο λειτουργικά καθώς συμβαίουν στο παρασκήνιο

- Χρήση λιγότερης ενέργειας, με αποτέλεσμα την μεγαλύτερη διάρκεια ζωής της μπαταρίας

- Microsoft Edge: δυνατότητα αλλάγης καρτελών σε κάθετη μορφή για ευκολότερη εύρεση συγκεκριμένης καρτέλας

- Για τα tablets με λειτουργικό σύστημα Windows 11 δυνατότητα γραφής με pen

- Για τα tablets με λειτουργικό σύστημα Windows 11 Kαινούριο πληκτρολόγιο με δυνατότητα αλλαγής θέματος, με emojis κ.α.

- Widgets με υλικό. Ειδήσεις, ημερολόγιο, καιρό, και οτι άλλο θέλετε να προσθέσετε και να τοποθετήσετε όπως σας βολεύει.

- Βελτιωμένο Microsoft Store με εφαρμογές από Android λογισμικό μέσω amazon appstore, ταινίες και σειρές από όλες τις πλατφόρμες μαζεμένες εδώ

- Μέσω Cloud της Microsoft(OneDrive) συνέχεια οποιασδήποτε διεργασίας ακριβώς από εκεί που έμεινε, σε οποιαδήποτε συσκευή ή πλατφόρμα εκτελούταν, χρησιμοποιώντας την ενότητα προτεινόμενα στο Μενού Έναρξης των Windows 11

-'Ασυναγώνιστη εμπειρία παιχνιδιού'

- DirectX 12 Ultimate

- Auto HDR (Προσαρμόζει τους φωτισμούς και τα χρώματα για ομορφότερο και ζωντανότερο αποτέλεσμα)

- Xbox Games Pass App ενσωματωμένο

Για υπολογιστές κορυφαίων επιδόσεων που τρέχουν Windows 11

- Παιχνίδια φορτώνουν γρηγορότερα, χάρεις την καινούρια τεχνολογία DirectStorage

- Το περιεχόμενο θα φορτώνει χωρίς την χρήση του επεξεργαστή, μόνο της κάρτας γραφικών

- Απίστευτα λεπτομερής κόσμοι παιχνιδιού που φορτώνουν αρκετά γρήγοροτερα

- Windows 11 PC's θα είναι έτοιμα και σε κομμάτι υλικού και σε κομμάτι οδηγών για να ενεργοποιήοσουν αυτή την τεχνολογία
